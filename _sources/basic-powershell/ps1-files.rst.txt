Speichern von Skripten
======================

Eine .ps1 (PowerShell Script Datei) dient dazu um mehrfach benötigte
Befehle und Funktionen zu speichern und einfach erneut auszuführen.

Ein PowerShell Script kann ganz einfach über die Powershell ISE
geschrieben und abgespeichert werden.

|image0|

Eine Variante ein bereits gespeichertes PowerShell Script kann nun per
Rechtsklick und anschließend mit “Mit PowerShell ausführen” ausführen.

|image1|

Eine andere Variante wäre das PowerShell Script im Scripteditor
“PowerShell ISE” über das Menü “Datei” und anschließend mit der Auswahl
“Öffnen…” Öffnen und im Editor ausführen.

|image2|

Noch eine Variante ein PowerShell Script auszuführen wäre einfach den
Dateinamen mit dem dazugehörigen Pfad in PowerShell einzugeben und mit
der Eingabetaste (Enter) auszuführen.

|image3|
